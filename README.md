## 💻 Getting Started <a name="getting-started"></a>

PR back-end : https://github.com/jdbs9514/hello-rails-back-end/pull/1


To get a local copy up and running, follow these steps.

### Prerequisites

You need to run the Rails app first:

https://github.com/jdbs9514/hello-rails-react-webpack.git


```sh
$ git clone
$ bundle install
$ rails server
```

after that you need to run the react app:

https://github.com/jdbs9514/hello-react-front-end.git
 
```sh
$ git clone
$ npm install
$ npm start
```

### Usage

once you run the rails and react app, go to react server and put this route:

http://localhost:3000/api/random_greeting
